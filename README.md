# Retail sales KPI Dasboard (SQL + Power BI)

## OVERVIEW
This project analyses over 500k+ retail transactions from a kaggle dataset and builds a KPI and returns risk dashboard.
This project aims to quantify revenue performance and identify returns related revenue losses and identify high returned products.

## TOOLS
- PostgreSQL
- Power BI
- SQL
